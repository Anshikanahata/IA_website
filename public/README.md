# Public Assets Folder

Place your assets here:

## Logo
- Add your `logo.png` or `logo.svg` file here
- Recommended size: 200x60px or similar aspect ratio
- Supported formats: PNG, SVG, JPG

## Modern India Images
- Add your modern India pictures here (e.g., `india-1.jpg`, `india-2.jpg`, etc.)
- Recommended: 3-5 images for the header gallery
- Formats: JPG, PNG, WebP
- Suggested dimensions: 1920x400px or similar wide format

## Usage
Files in this folder can be referenced directly:
- Logo: `/logo.png` or `/logo.svg`
- Images: `/india-1.jpg`, `/india-2.jpg`, etc.
